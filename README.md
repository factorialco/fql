# FQL (Factorial Query Language)

A flexible, extensible query language that compiles down to Arel (to extend queries within a Rails application) or plain Ruby.

## Quick start

In your Gemfile:

```
gem 'fql', git: 'git@github.com/factorialco/fql'
```
